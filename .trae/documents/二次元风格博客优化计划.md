# 二次元风格博客优化计划

## 1. 主题风格优化
- **更换主题风格**：将NexT主题的scheme从Muse改为Gemini，更适合搭配二次元元素
- **调整配色方案**：修改主题颜色为明亮活泼的二次元色调
- **添加自定义CSS**：创建source/_data/styles.styl文件，添加动漫背景、渐变效果等二次元样式

## 2. 动态元素添加
- **启用背景粒子效果**：配置canvas_nest或其他背景粒子插件
- **添加鼠标特效**：启用点击特效和鼠标跟踪效果
- **增强页面动画**：优化现有motion动画，添加更多过渡效果

## 3. Live2D角色配置
- **安装插件**：`npm install hexo-helper-live2d --save`
- **配置角色模型**：在_config.yml中添加live2d配置，选择合适的二次元角色（如hijiki、tororo等）
- **调整角色设置**：设置角色位置、大小和交互效果

## 4. 日夜模式切换
- **启用lightdark模式**：在themes/next/_config.yml中配置lightdark选项
- **配置切换方式**：支持自动切换和手动切换
- **优化模式适配**：确保所有元素在不同模式下都能正常显示

## 5. 其他优化
- **添加看板娘**：配置live2d角色的交互功能
- **优化字体**：引入适合二次元风格的字体
- **调整布局**：优化页面布局，增强视觉层次感

所有配置都将通过修改配置文件实现，无需修改主题核心代码，确保升级兼容性。
